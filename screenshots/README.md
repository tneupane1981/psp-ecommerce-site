# Screenshots

This folder contains screenshot images used in the **Project Report** (`report.html`).

## How to Add Screenshots

Open each page in your browser, take a screenshot, and save the file to this folder using the exact filename listed below.

| Filename | Page to Capture | Description |
|---|---|---|
| `concept-page.png` | `concept.html` | Full view of the Conceptual Site Overview page |
| `wireframes-page.png` | `wireframes.html` | Full view of the Wireframe Mockups page showing layout diagrams |
| `common-header-nav.png` | Any page | The consistent site header, logo, and navigation bar |
| `common-footer.png` | Any page | The consistent site footer (scroll to bottom) |
| `common-breadcrumbs.png` | Any interior page | Breadcrumb navigation trail below the nav bar |
| `sitemap-page.png` | `sitemap.html` | Full view of the Sitemap page showing all page links |
| `css-variables.png` | `css/styles.css` | The `:root` CSS custom properties block in a code editor or browser DevTools |

## Tips

- Use a browser width of **1280px** for consistent desktop-size screenshots.
- Tools you can use: browser built-in screenshot (F12 DevTools → capture full page), or any screen-capture utility.
- PNG format is preferred. JPEG is also acceptable — just update the `<img src>` paths in `report.html` accordingly.
